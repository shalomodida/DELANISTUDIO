Quick-sign is a system build for automating the processing of attendance to enhance the speed of performing attendance task easily. The system will also help in evacuating attendance eligibility criteria of an employee, it will also falicitate to access the attendance information of aparticular employee in an organisation.

User Guide.

This site allows employees to easily check-in, check-out and also view shared documents in their computers.

Admin(HR)

The admin creates account and add employees into the attendance system.
Admin issues employees with username and password where the default username and password are employee number and identity number respectively.
When the admin logs-in he/she can view employee check-in and check-out, and post the notifications to the employees.
